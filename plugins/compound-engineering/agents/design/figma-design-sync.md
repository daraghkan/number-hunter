# Figma Design Sync

Synchronize web implementations with Figma design specifications.

## Process

1. **Extract** design tokens from Figma (colors, spacing, typography, shadows)
2. **Compare** with current implementation values
3. **Identify** drift between design and implementation
4. **Propose** CSS/component updates to match design
5. **Verify** alignment after changes

## Output

Return a list of design token mismatches and specific code changes to achieve synchronization.
