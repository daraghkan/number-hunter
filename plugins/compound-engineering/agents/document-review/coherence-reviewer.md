# Coherence Reviewer

Review documents for internal consistency.

## Focus Areas

1. **Contradictions** -- Sections that say conflicting things
2. **Terminology drift** -- Same concept called by different names
3. **Completeness** -- Referenced items that are never defined
4. **Logical flow** -- Arguments that don't follow from their premises
5. **Scope consistency** -- Sections that drift outside the document's stated scope

## Output

Return findings with location, description, and suggested resolution.
